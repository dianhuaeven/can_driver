### MtCan 关键验证点（建议回归）

1. 发送 `requestState(1)`，确认 `0x241` 响应更新 `motorStates[1]`。  
2. 调用 `getCurrent(1)/getVelocity(1)`，确认能回读非零状态。  
3. `Disable(1)` 后验证发送命令为 `0x80`。  
4. 连续 `Enable/Disable` 循环，确认不会反复写零点和复位。  
5. 运动过程中连续读取 `getPosition`，确认是实际位置变化而非目标值瞬时跳变。
